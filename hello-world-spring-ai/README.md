# Hello World Spring AI

Minimal Spring Boot sample generated from Spring Initializr.

## Dependencies

- Spring Web
- Spring Data JPA
- H2 Database
- Spring AI OpenAI starter
- Spring Boot Test

## Run

```bash
./mvnw spring-boot:run
```

Then call:

```bash
curl http://localhost:8080/
```

Expected response:

```text
Hello, World!
```
